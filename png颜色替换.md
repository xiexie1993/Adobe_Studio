# 修改png颜色

## 用图层叠加修改

+ 前提：Adobe Photoshop CC 2018

+ 图层颜色叠加
    + step1: 工具栏 --> 打开 ---> 选择文件
    + step2: 选择好要修改的图层选中（右下角图层窗口）
    + step2: 加载图片后、菜单栏 选择--> 图层--> 图层样式 --> 颜色叠加
    + step3: 取样颜色，填写颜色值，透明度等参数，查看预览
    + step4: 点击确定，导出图片
