# 抠图教程

## 构图步骤
+ 前提：Adobe Photoshop CC 2018

+ 抠图章
    + step1: 工具栏 --> 打开 ---> 选择要抠的文件
    + step2: 加载图片后、菜单栏 选择--> 色彩范围
    + step3: 取样颜色，填写颜色容差值
    + step4: 点击确定，生成抠图范围,
    + step5: 菜单栏 选择--> 选择是否要反选
    + step6: 进行反选，只取公章的红色区域
